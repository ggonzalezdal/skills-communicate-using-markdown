### Code Example
#### C# Fibonacci Sequence 
``` C#
long a = 0;
long b = 1;

Console.WriteLine($"\nFibonacci sequence for {count} numbers:");

for (int i = 0; i < count; i++)
{
    Console.Write(a + " ");
    long next = a + b;
    a = b;
    b = next;
}
Console.WriteLine();
```
